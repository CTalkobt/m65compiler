# Mega65 C Compiler (cc45)

## Overview
The `cc45` tool translates C source code into 45GS02 assembly for use with `ca45`.

## Supported Features
- **Functions**: Translated to `PROC` / `ENDPROC` blocks.
- **Parameters**: `int` parameters are treated as 16-bit words (`W#`).
- **Return**: `return <expr>` emits `ENDPROC`.
- **Function Calls**: Translated to `CALL`.
- **String Literals**: Automatically pooled into a data section using `.text` (PETSCII).

## Usage
```bash
cc45 [-v] [-c] [-o output.s] <input.c>
```
- `-v`: Verbose (prints tokens, AST, and generated assembly).
- `-c`: Calls `ca45` to produce a `.bin` file.

## Implementation
- **Lexer**: C-style tokens + comment stripping.
- **Parser**: Recursive descent producing an AST.
- **Code Generator**: Visitor-based traversal emitting `ca45` assembly.
