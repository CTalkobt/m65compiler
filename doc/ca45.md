# Mega65 Assembler (ca45)

## Overview
The `ca45` tool is an assembler for the Mega65 45GS02 CPU. It supports standard 6502/45GS02 opcodes along with extended macro-like opcodes for procedure handling.

## Syntax
- **Comments**: Start with a semicolon (`;`).
- **Labels**: `name:`
- **Equates**: `NAME = value` (supports `$`, `%`, `'A'`, decimal).
- **Directives**:
    - `.org <addr>`: Set Program Counter.
    - `.byte <v1>, <v2>...`: Emit 8-bit bytes.
    - `.word <v1>, <v2>...`: Emit 16-bit words (little-endian).
    - `.text "<string>"`: Emit PETSCII string (case-swapped).
    - `.ascii "<string>"`: Emit raw ASCII string.

## Extended Opcodes
- `CALL <name>[, <arg1>, <arg2>...]`:
    - Pushes arguments onto stack.
    - Uses `B#` or `W#` prefixes to force size, otherwise infers from symbol table.
    - Pushes in order provided; last argument is closest to SP after JSR.
- `PROC <name>[, [B#|W#]<argname>...]`:
    - Defines a procedure and its named parameters.
    - Maps `argname` to the correct stack offset (`offset, s`).
- `ENDPROC`:
    - Emits `RTN #<total_param_size>` to clean stack and return.

## Implementation Details
- **Pass 1**: Collects symbols, calculates sizes, and maps procedure contexts.
- **Pass 2**: Emits binary encoding and performs PETSCII conversion for `.text`.
